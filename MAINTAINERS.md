## Approval Model

This is a single-maintainer project (BDFL: Benevolent Dictator For Life). All PRs require [@FarzamHejaziK](https://github.com/FarzamHejaziK)'s approval before merge, enforced via [`.github/CODEOWNERS`](.github/CODEOWNERS) plus branch protection on `main`.

Direct pushes to `main` are blocked. Force-push and branch deletion are blocked.

## Becoming a Maintainer

Out of scope for v0. If the project grows enough that a co-maintainer is justified, we'll add a [`GOVERNANCE.md`](GOVERNANCE.md) covering the process. For now: contribute via PRs, build a track record, and we'll talk if it's a good fit.
